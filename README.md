# SmartInvoice Pro (SI-PRO) — Page Vitrine & Showcase 🚀

> **Projet de Fin d'Études (PFE) 2026**
>
> SmartInvoice Pro est une solution SaaS moderne de facturation conçue pour aider les freelances et les petites entreprises à créer, envoyer et suivre leurs factures en toute simplicité.

## ✨ Fonctionnalités Clés

1. **🔒 Authentification Sécurisée** — Système complet d'inscription, de connexion et de protection des routes via Laravel Sanctum.
2. **📄 Factures PDF** — Génération automatisée et propre de factures au format PDF à l'aide de DomPDF.
3. **📧 Envoi Automatique par Mail** — Transmission instantanée des factures générées aux clients via un service d'envoi SMTP.
4. **📊 Dashboard Financier** — Statistiques détaillées en temps réel (chiffre d'affaires, statut des factures, progression mensuelle, etc.).
5. **💎 Modèle Freemium & Pro** — Gestion fine des quotas de factures gratuites et options de mise à niveau premium.
6. **👥 CRM Simple** — Gestion des dossiers clients, de l'historique de facturation et des informations de contact.

## 🛠️ Stack Technique

Le projet repose sur une architecture moderne, fluide et robuste :

### Frontend
- **React 19** & **Vite** — Rapidité de build et réactivité de l'interface.
- **Tailwind CSS v4** — Design épuré, responsive et totalement personnalisé.
- **Framer Motion** — Micro-animations et transitions fluides de qualité premium.
- **Lucide React** — Set d'icônes vectorielles modernes et légères.

### Backend
- **Laravel 11** — Framework PHP puissant pour l'API REST.
- **Laravel Sanctum** — Gestion robuste des jetons d'authentification API.
- **MySQL** — Base de données relationnelle sécurisée.
- **DomPDF** — Librairie PHP de conversion HTML vers PDF.
- **SMTP Server** — Passerelle sécurisée d'expédition d'emails.

## ⚙️ Démarrage Local de la Page Vitrine

Pour installer et exécuter localement la page vitrine (showcase) :

### 1. Cloner le dépôt
```bash
git clone https://github.com/solaymanelkharraz/showcase.git
cd showcase
```

### 2. Installer les dépendances
```bash
npm install
```

### 3. Lancer le serveur de développement
```bash
npm run dev
```

L'application sera accessible par défaut sur `http://localhost:5173`.
